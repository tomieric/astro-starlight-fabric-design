---
title: 设计价值观
---

> Fabric Design 为产品提供一致的用户体验。通过创建清晰、高效、包容、统一的设计原则，提升产品体验，并辅助用户达成业务目标。


import { Card, CardGrid } from '@astrojs/starlight/components';

<CardGrid>
  <Card title="清晰" icon="star">
    每个页面都应有明确的重点，以便于用户每进入一个新页面的时候都能快速地理解页面内容。在确定了重点的前提下，应尽量避免页面上出现其它与用户的决策和操作无关的干扰因素。
  </Card>
  <Card title="高效" icon="moon">
    我们希望产品能助力用户高效地完成工作任务。通过视觉元素引导、交互控件有序排列、状态及时反馈等方式提升用户的工作效率。
  </Card>
  <Card title="包容" icon="document">
    Fabric Design 伴随着行业内用户的使用后获得行业的反哺，将会不断得到多元内容补充。在保证设计价值观一致的前提下，以开放创新的态度洞察多个实际场景需求，赋能纺织行业 ，为科技纺织探索更多的多元化机会点。
  </Card>
  <Card title="统一" icon="sun">
    样式规范、操作流程、交互方式保持协调一致, 这样统一的设计标准, 能够体现产品的品牌感与信赖感。同时协调一致的重复，能降低用户反复学习成本，进而提升其效率。
  </Card>
</CardGrid>
