在 `npm` 我们可以找到 `qb-ui` 的安装包

## 使用

```bash
pnpm add qb-ui
```

在入口文件引入：

```js
import Vue from 'vue'
import QBUI from 'qb-ui'

Vue.use(QBUI)
```

在页面文件中使用：

```vue
<template>
  <qb-button type="primary" @click="handleClick">button</qb-button>
</template>

<script>
export default {
  methods: {
    handleClick() {
      console.log('click!')
    }
  }
}
</script>
```
